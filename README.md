# 🏠 House Price Prediction using Machine Learning

## 📌 Overview
This project predicts house prices using machine learning based on multiple features such as area, number of rooms, and age of the house.

## 🎯 Objective
To build a predictive model that estimates house prices accurately.

## ⚙️ Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## 🧠 Model Used
Linear Regression

## 📊 Features
- Area (sq ft)
- Number of rooms
- Age of house

## 📈 Output
The model predicts the price of a house based on input features.

## 🚀 How to Run
1. Install required libraries:
   pip install -r requirements.txt

2. Run the notebook:
   house_price.ipynb
